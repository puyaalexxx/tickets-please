# TicketsPlease - Laravel API Project

## Features:

- Laravel 12
- PHP 8.3
- Routing
- Models
- Middleware
- Controllers
- Mysql Database
- Migrations
- Factories for dummy data
- Database seeding
- Model Relationships
- Api Endpoints
- Api Versioning
- Sanctum for Token Authentication
- Api Resources for formatted responses
- Custom Requests
- Validation
- Filtering (tickets)
- Custom Query Builder
